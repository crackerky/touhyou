# touhyou

ウォレット認証投票システム - Cardanoブロックチェーンを使った分散型投票アプリケーション

## 🚀 主な機能

- 🔐 Cardanoウォレット認証による本人確認
- 📊 リアルタイム投票・結果表示
- 🎨 モダンUI（React + TypeScript + TailwindCSS）
- 🔄 重複投票防止システム

## 🛠️ 技術スタック

- **フロントエンド**: React 18, TypeScript, Vite
- **ウォレット連携**: MeshSDK (Cardano)
- **データベース**: Supabase
- **スタイリング**: TailwindCSS, Framer Motion
- **状態管理**: Zustand

## 📋 必要な環境

- Node.js 18+ 
- npm または yarn
- Supabaseアカウント
